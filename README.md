# TriangleProduct - Avalonia

Проект на C# и Avalonia для определения типа треугольника.

## Что делает программа

- принимает три значения сторон;
- проверяет корректность ввода;
- определяет тип треугольника;
- выводит понятное сообщение об ошибке или результате.

## Встроенные механизмы защиты

- проверка на пустые поля;
- проверка, что введены целые числа;
- проверка на ноль и отрицательные значения;
- проверка неравенства треугольника.

## Структура

- `TriangleProduct` - основное Avalonia-приложение;
- `TriangleProduct.Tests` - автоматические тесты для логики проверки.

## Как запустить

1. Установить .NET SDK 8 или новее.
2. Установить шаблоны Avalonia при необходимости:
   `dotnet new install Avalonia.Templates`
3. Открыть папку в Rider или в другом редакторе кода.
4. Выполнить:
   - `dotnet restore`
   - `dotnet run --project TriangleProduct`

## Как запустить тесты

```bash
dotnet test
```
